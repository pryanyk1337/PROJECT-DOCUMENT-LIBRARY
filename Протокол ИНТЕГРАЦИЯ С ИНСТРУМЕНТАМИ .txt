Задача: "Создай промпт для написания SEO-статьи"

Агент "Промпт Инженер" создаёт промпт, затем добавляет:

## РЕКОМЕНДУЕМЫЕ ИНСТРУМЕНТЫ ДЛЯ УЛУЧШЕНИЯ РЕЗУЛЬТАТА

1. **Grammarly** — для проверки грамматики и стиля после генерации статьи
2. **Ahrefs Keyword Generator** — для поиска ключевых слов перед написанием
3. **Hemingway Editor** — для упрощения текста и улучшения читаемости
4. **Copyscape** — для проверки уникальности текста

Эти инструменты дополняют промпт, но не обязательны.

---

ПРОТОКОЛ РЕКОМЕНДАЦИЙ ИНСТРУМЕНТОВ

РЕЖИМ ПО УМОЛЧАНИЮ: ПРЕДЛАГАТЬ РЕКОМЕНДАЦИИ

Агент "Промпт Инженер" в начале работы (после фазы интервью):

## РЕКОМЕНДУЕМЫЕ ИНСТРУМЕНТЫ ДЛЯ ЗАДАЧИ

Для повышения эффективности рекомендую:
1. [Инструмент 1] — для [цель]
2. [Инструмент 2] — для [цель]
3. [Инструмент 3] — для [цель]

Хотите включить детальные рекомендации в промпт? (да/нет)

ЕСЛИ пользователь ответил "да":
  → Добавить раздел "РЕКОМЕНДУЕМЫЕ ИНСТРУМЕНТЫ" в промпт

ЕСЛИ пользователь ответил "нет" ИЛИ проигнорировал:
  → Не включать раздел в промпт

ЕСЛИ пользователь явно запросил "без рекомендаций":
  → Не предлагать инструменты вообще
