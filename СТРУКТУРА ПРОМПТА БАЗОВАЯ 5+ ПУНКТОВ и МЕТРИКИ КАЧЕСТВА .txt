УРОВЕНЬ 1: КОНТЕКСТ
├─ Цель задачи
├─ Целевая аудитория результата
└─ Ограничения (время, формат, объём)

УРОВЕНЬ 2: РОЛЬ (ИДЕНТИЧНОСТЬ)
├─ Кто агент (эксперт в чём)
├─ Ключевые навыки (3-5)
└─ Тон общения

УРОВЕНЬ 3: ФАЗА ИНТЕРВЬЮ (ОБЯЗАТЕЛЬНА НА НАЧАЛЬНОМ ЭТАПЕ)
├─ Применить фильтры NLP метамодели (упущения, обобщения, причинность)
├─ Задать 4-6 уточняющих вопросов
├─ Сформулировать явные предположения при неопределённости
└─ Получить подтверждение для продолжения

УРОВЕНЬ 4: ИНСТРУКЦИИ (ОСНОВНАЯ РАБОТА)
├─ Пошаговый алгоритм (нумерованный)
├─ Самопроверка на промежуточных шагах (внутренняя, без вывода в чат)
└─ Автономные механизмы решения проблем

УРОВЕНЬ 5: ОГРАНИЧЕНИЯ
├─ Что НЕ делать (запреты)
├─ Граничные условия (edge-кейсы)
└─ Критерии качества

УРОВЕНЬ 6: САМОПРОВЕРКА ПЕРЕД ФИНАЛЬНОЙ ВЫДАЧЕЙ
├─ Чек-лист критериев (внутренняя проверка)
├─ Критерии соответствия запросу пользователя
└─ Если не соответствует → доработка → повторная проверка

УРОВЕНЬ 7: ФОРМАТ ВЫХОДА
├─ Структура результата (текст/таблица/JSON/код)
├─ Обязательные элементы
└─ Пример готового результата


Автомодульность если >2500:

МОДУЛЬ 1: БАЗОВЫЙ ПРОМПТ (1500 слов)
- Роль и контекст
- Основные инструкции
- Формат выхода

МОДУЛЬ 2: РАСШИРЕННЫЕ ИНСТРУКЦИИ (1000 слов)
- Детальные алгоритмы
- Обработка edge-кейсов

МОДУЛЬ 3: ДОПОЛНИТЕЛЬНЫЕ ФУНКЦИИ (500 слов)
- Опциональные возможности

ЦЕЛЕВАЯ ДЛИНА ПРОМПТА: 1500-2000 слов (оптимум)

ЕСЛИ задача простая → промпт 800-1200 слов
ЕСЛИ задача средняя → промпт 1500-2000 слов
ЕСЛИ задача сложная → промпт 2000-2500 слов

ЕСЛИ промпт получается >2500 слов:
→ Разбить на БАЗОВЫЙ (обязательный) + РАСШИРЕННЫЕ МОДУЛИ (опциональные)



ПРОВЕРКА ПРОМПТА ПЕРЕД ВЫДАЧЕЙ:

□ 1. ПОЛНОТА СТРУКТУРЫ
   Все 7 уровней присутствуют (контекст, роль, интервью, инструкции, ограничения, самопроверка, формат выхода)?

□ 2. ОДНОЗНАЧНОСТЬ ФОРМУЛИРОВОК
   Нет двусмысленных терминов? Проверено через NLP метамодель (упущения, обобщения)?

□ 3. ИЗМЕРИМОСТЬ КРИТЕРИЕВ
   Критерии успеха определены? Можно проверить выполнение?

□ 4. АВТОНОМНОСТЬ
   Инструкции для ИИ-агента, НЕ для пользователя? Агент может выполнить задачу самостоятельно?

□ 5. ФАЗА ИНТЕРВЬЮ
   Включена обязательная фаза интервью с 4-6 вопросами?

□ 6. ГРАНИЧНЫЕ УСЛОВИЯ
   Описано поведение агента при проблемах (что делать если данных нет / задача неясна / ошибка)?

□ 7. ЗАПРЕТЫ
   Явно указано, что агент НЕ должен делать (для предотвращения галлюцинаций/ошибок)?

□ 8. ФОРМАТ ВЫХОДА
   Чётко описан ожидаемый результат? Есть пример?

□ 9. АДАПТАЦИЯ ПОД ЦА
   Учтена целевая аудитория результата (тон, стиль, глубина)?

□ 10. ТОКЕН-ЭФФЕКТИВНОСТЬ
   Нет избыточных повторов? Каждая инструкция необходима?
