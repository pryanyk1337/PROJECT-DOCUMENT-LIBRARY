УРОВЕНЬ 1: ОПРЕДЕЛЕНИЕ ЯЗЫКА ПОЛЬЗОВАТЕЛЯ
├─ Автоопределение по первому сообщению
└─ Опциональный запрос: "На каком языке продолжить?"

УРОВЕНЬ 2: ЯЗЫК ДИАЛОГА
├─ Стандарт: язык пользователя
├─ Исключение: пользователь запросил другой язык явно
│   Пример: русскоязычный пользователь создаёт англоязычный контент
└─ Гибкость: переключение языка по запросу

УРОВЕНЬ 3: ЯЗЫК ПОИСКА ИНФОРМАЦИИ
├─ ВСЕГДА минимум 2 языка: Английский + Русский
├─ Английский → международные данные, научные статьи, новые тренды
├─ Русский → международные данные (СНГ), локальные источники, региональная специфика
└─ Дополнительные языки: по запросу пользователя

ПОПРАВКА: СТАТУС РУССКОГО ЯЗЫКА
Русский язык = МЕЖДУНАРОДНЫЙ язык (страны СНГ: Украина, Беларусь, Россия, Казахстан, Киргизия и др.)
НЕ имеет исключительного отношения к РФ.
Используется для поиска данных по региону СНГ и русскоязычным источникам.

УРОВЕНЬ 4: ЯЗЫК ВНУТРЕННИХ РАССУЖДЕНИЙ (НОВОЕ)
Вопрос: НА КАКОМ ЯЗЫКЕ НЕЙРОСЕТЬ ЭФФЕКТИВНЕЕ РАССУЖДАЕТ?

Анализ по платформам:
├─ GPT-4/5: Английский (обучена преимущественно на английском корпусе)
├─ Claude: Английский (аналогично)
├─ Gemini: Английский (Google, англоязычная база)
├─ Deepseek: Китайский + Английский (китайская разработка)
└─ Grok: Английский (X/Twitter база)

ПРАВИЛО ДЛЯ АГЕНТА "ПРОМПТ ИНЖЕНЕР":
IF (задача требует глубокого анализа ИЛИ сложных рассуждений):
  → Инструкция в промпте: "Рассуждай внутренне на английском (если платформа лучше работает на EN)"
  → НО финальный результат → на языке пользователя

ЕСЛИ (платформа одинаково эффективна на обоих языках):
  → Рассуждения на языке пользователя

УРОВЕНЬ 5: ЯЗЫК ФИНАЛЬНОЙ ВЫДАЧИ
├─ Стандарт: язык пользователя
├─ Исключение 1: задача явно требует другой язык
│   Пример: "Напиши статью на английском"
├─ Исключение 2: специфика платформы/инструмента
│   Пример: GitHub Spaces инструкции → рекомендован английский
└─ Гибкость: переключение по запросу

ПРОВЕРКА МУЛЬТИЯЗЫЧНОСТИ (чек-лист):
□ Поиск информации проведён минимум на 2 языках (EN + RU)?
□ Язык диалога соответствует языку пользователя (или запросу)?
□ Язык рассуждений оптимален для платформы?
□ Язык результата соответствует задаче?
□ Терминология корректна на целевом языке?
